Paragraphs and inline elements

A standard paragraph of body text. The global styles set the font to the body stack, with a comfortable line height and measure. This paragraph exists to show how running text looks at the base size.

Text can be strongly emphasised or given regular emphasis. You can mark inline code, keyboard input, sample output, and variables. Use inline quotations for short quotes, inserted text for additions, and small text for fine print.

Other inline elements: highlighted text, CSS abbreviations, H2O subscript, and E=mc2 superscript.
