A Prose Example

The prose block configures flow spacing for long-form content. It does not do layout itself — it sets --flow-space for different element types within a flow composition.

Paragraphs are constrained to --measure width and use text-wrap: pretty for better line breaking.

Subheading with tighter follow-on spacing

The element immediately after a heading gets reduced spacing via --space-m instead of the default --space-l.

Lists

  • List items have tight spacing between them.
  • Nested lists are also tight.
  • This keeps lists compact and readable.
  1. Ordered lists behave the same way.
  2. Consistent vertical rhythm throughout.

Code

Inline code like var(--flow-space) is styled by global styles. Code blocks get extra vertical space:

function greet(name) {
  return `Hello, ${name}!`;
}

The paragraph after a code block also gets extra space.
